使用nginx命令需要进入到nginx可执行文件目录 `/usr/sbin/`

[nginx document](http://nginx.org/en/docs/)

#### 查看版本

```
nginx -v
```

#### 启动nginx

```
./nginx
```

#### 快速关闭

```
nginx -s stop
```

#### 安全关闭

```
nginx -s quit
```

#### 重新加载配置文件

```
nginx -s reload
```
