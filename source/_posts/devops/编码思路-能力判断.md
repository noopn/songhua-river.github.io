#### 是否是浏览器环境

```js
// 使用 typeof, 对于不存在的变量会返回 undefined
const hasDocument = typeof document !== "undefined";
```

#### 检查 IE 浏览器

```js
var isIE11 =
  typeof navigator !== "undefined" &&
  navigator.userAgent.indexOf("Trident") !== -1;
```
